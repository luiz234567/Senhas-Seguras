# Senhas-Segura

Este projeto foi desenvolvido com o objetivo de conscientizar as pessoas sobre a importância da segurança digital por meio da criação de senhas fortes. O sistema permite gerar senhas seguras, verificar o nível de segurança de uma senha e mostrar quais requisitos foram atendidos, como o uso de letras maiúsculas e minúsculas, números, caracteres especiais e um tamanho adequado.

Além disso, o projeto possui um botão para mostrar ou ocultar a senha, copiar a senha gerada e um gráfico que representa visualmente o nível de segurança. A interface foi criada com cores claras, tornando a navegação simples, agradável e intuitiva.

O principal objetivo deste trabalho é incentivar boas práticas de proteção de dados e demonstrar, de forma prática, como a programação pode contribuir para aumentar a segurança das informações no ambiente digital.
